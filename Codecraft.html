Python Tutorials

Explore my Python tutorials to learn basic and advanced concepts:

# Basic Syntax
print("Hello, World!")

# Variables and Data Types
x = 5  # Integer
y = 3.14  # Float
name = "Suryadev"  # String
numbers = [1, 2, 3, 4, 5]  # List

# Control Flow
if x > 0:
    print("x is positive")

for number in numbers:
    print(number)

count = 0
while count < 5:
    print(count)
    count += 1

# Functions
def greet(name):
    return f"Hello, {name}!"

message = greet("Suryadev")
print(message)
